# 문자열 정리

파이썬 문자열을 정리해보자

<br>

## 문자열 인덱싱

```
>>> a = "Hello World!"
>>> len(a)
12
```

```len()```은 문자열을 측정하는 함수이다.

```
>>> a[0]
'H'
>>> a[1]
'e'
>>> a[2]
'l'
>>> a[3]
'l'
>>> a[5]
' '
>>> a[-1]
'!'
```

문자열을 뒤에서도 읽을 수 있다. 그러나 -13 < < 13 까지만 가능.

<br>

## 문자열 슬라이싱

```
>>> a = "Hello World!"
>>> a[0:5]
'Hello'
```

```a[0:5]```는 인덱스 0부터 5까지의 문자열을 슬라이싱하는 역할을 한다. 끝번호인 '5' 는 해당되지 않는다.
